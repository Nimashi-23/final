Home page

Technical Description

The home page was done in HTML, CSS. Semantic HTML elements like header, nav,cards and footer were used to create a structured layout. It is a common best practice to use an unordered list for a navigation menu — and it can be stylised with CSS to be user friendly. To create a background video used video tag with autoplay, loop and mute functions.

Accessibility

Annotations were made through ARIA attributes to enhance screen reader accessibility. We used proper color contrast, all images had descriptive alt text. The organization of the system was done clearly to allow for easy access for users with disabilities.

Table Page

Technical Description

The th, tr, td, and br tags were used to define the table page to make the data more readable. CSS was used to style the table to add borders, preferable padding, column background colors to make it more legible. Give it some hover effect so that it is easier to read.

Accessibility

'Table' tags were correctly implemented by using 'th' tag which gave a better context to screen readers. READABILITY was ensured with adequate contrast. Kept the same format and added some icons in the table head using box-icon and images to implement the each content

Technical Description

Headings and paragraphs were used to format the content page for ease of reading. We implemented an internal linking system to help users quickly reach various sections. To improve user experience a button 'Go to Top' was acquired.

Challenges and Lessons Learned

Compliance

No sensitive or personal information is gathered or kept without the user's permission. Users are not tracked by the website beyond what is required for basic operation. Every type of media, including pictures and videos, is either unique, free of royalties, or correctly credited to its original creator. When required, content is properly cited. The information is presented clearly and has been fact-checked. There isn't any dangerous, deceptive, or misleading stuff.
